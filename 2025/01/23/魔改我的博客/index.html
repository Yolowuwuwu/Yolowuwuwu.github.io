魔改我的博客

|总字数:1.1k|阅读时长:4分钟|浏览量:

博客魔改参考：https://blog.anheyu.com/posts/sdxhu.html

1、加上侧边电子时钟


类似如图所示。
具体步骤如下：

1.1 安装下载库

npm install hexo-butterfly-clock-anzhiyu –save

1.2 添加配置项

添加到_config.butterfly.yml的文档末端

1.3 申请key

其中一下两个参数需要自己申请。

qweather_key: # 和风天气key
gaud_map_key: # 高得地图web服务key

1.3.1 申请qweather_key

1、进入链接： https://id.qweather.com/#/login
2、登录/注册之后，进入控制台，如图所示

3、选择创建项目（此处我已经创建过了，所以变成了管理）

4、名称随意取，选择免费订阅
5、KEY的类型选择Web API
6、直接将生成的key复制进去即可

1.3.2 申请gaud_map_key

1、进入链接：https://lbs.amap.com/
2、登录/注册后进入控制台，选择应用管理->我的应用->创建新应用

3、应用名称随意，应用类型选择其他，点击新建

4、点击添加KEY

5、KEY名称随意，服务平台选择Web服务，点击提交即可

6、复制KEY即可

1.4 部署到博客

2、添加留言板：薇尔莉特信封

2.1 安装库

npm install hexo-butterfly-envelope –save

2.2 添加配置项

添加到_config.butterfly.yml的文档末端

代码中的图片链接均可以根据自己喜好修改

2.3 在menu中添加

在_config.butterfly.yml的menu中添加如下：

留言板: /comments/ || fas fa-envelope

3、个人卡片渐变色

3.1 新建css文件

在blog文件夹中的source文件夹中新建文件夹css，在css文件夹中新建custom.css文件

3.2 复制代码

复制一下代码到自定义的css文件夹中；颜色根据自己喜好更改

3.3 在inject配置项引入

在_config.butterfly.yml中搜索inject，在head中添加

-< link rel=”stylesheet” href=”/css/custom.css”>

3.4 部署到自己的博客
